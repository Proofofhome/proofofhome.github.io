# ProofOfHome

Building sovereign technology one home at a time.

ProofOfHome documents experiments in Bitcoin, privacy, self-hosted technology, and resilient home infrastructure.

## Website

https://proofofhome.github.io/

## Current Experiment

### Experiment 001: Genesis Laptop

Can an old laptop become useful sovereign home infrastructure?

A Dell Latitude E7450 originally running Windows 7 is being repurposed into a Bitcoin home node.

## Project Structure

- `experiments/` - real-world hardware and software experiments
- `images/` - experiment photos
- `lab.md` - hardware and tools
- `roadmap.md` - future plans

## Mission

Explore practical ways ordinary households can build more independent, private, and resilient technology systems.
