# ProofOfHome Genesis: Day One

## Why This Project Exists

ProofOfHome started with a simple question:

Can an ordinary person build a more sovereign home using affordable technology?

Modern homes depend on many systems we don't fully understand:
- cloud services
- financial systems
- digital identity
- communications
- energy infrastructure

This project is about learning how these systems work and experimenting with alternatives.

## Why Start With Bitcoin?

The first experiment begins with Bitcoin because running your own node is a practical example of owning infrastructure.

Instead of only using a network, I want to understand what is required to participate directly.

## Season 1: Bitcoin Home Node

The first goal:

Build and operate a Bitcoin node at home.

I will document:

- Hardware choices
- Setup process
- Costs
- Power usage
- Problems
- Lessons learned

## The Rules

Every experiment will include:

- Real costs
- Real results
- Mistakes
- What worked
- What didn't
