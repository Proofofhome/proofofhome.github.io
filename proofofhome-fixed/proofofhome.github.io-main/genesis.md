## Why Start With Bitcoin?

Running your own node is a practical, concrete example of owning infrastructure rather than only using someone else's. Instead of just using the network, the first experiment is about understanding what it actually takes to participate in it directly.

## Season 1: Bitcoin Home Node

Goal: build and operate a Bitcoin node at home, using existing hardware where possible.

Documented along the way:

- Hardware choices
- Setup process
- Costs
- Power usage
- Problems
- Lessons learned
