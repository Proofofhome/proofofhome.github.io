# Roadmap

ProofOfHome moves in seasons. Each season is a real, documented experiment — not a plan that stays theoretical.

## Season 1 — Bitcoin Foundation

- Build the first Bitcoin home node
- Document hardware choices and setup
- Measure costs, sync time, and power usage

## Season 2 — Sovereign Home Server

- Self-hosted services (files, backups, dashboards)
- Local-first alternatives to cloud services
- Home monitoring

## Season 3 — Physical Builds

- Bitcoin displays and electronics projects
- Sensors and monitoring hardware
- Hands-on repair and hardware reuse

## Season 4 — Energy Independence

- Solar integration
- Battery and backup power
- Real power-usage measurement across the lab

## Season 5 — Practical Resilience

- Backup power for essential loads
- Water storage and basic filtration
- Food storage basics
- Off-grid communications

Each season builds on the last. Nothing moves to a new season until the current one has real, published results — not just a plan.
