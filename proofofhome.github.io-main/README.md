# ProofOfHome

Building capable homes in a digital world.

ProofOfHome is an open-source, pseudonymous project documenting one person's
real attempt to build sovereign, private, resilient home technology —
starting with a Bitcoin node, and expanding into self-hosting, energy
independence, and practical resilience.

## Website

https://proofofhome.github.io/

## Current Experiment

### Experiment 001: Genesis Laptop

Can an old laptop become useful sovereign home infrastructure?

A Dell Latitude E7450 originally running Windows 7 is being repurposed into a Bitcoin home node.

## Mission

Explore practical ways ordinary households can build more independent, private, and resilient technology systems — with real hardware, real costs, and real failures documented along the way.
